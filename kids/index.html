🍏 macOS Speech Fixing Instructions

If you only hear the drum pops but no voice reading, the macOS speech synthesis daemon might be locked up. Copy and run this command in your Mac's Terminal application to restart it:

killall SpeechSynthesisServer com.apple.speech.speechsynthesisd
